Реальный счёт T-Bank — депозит и торговля не подключено Развернуть

Токен нужен для чтения счёта и депозита; для режима «Реальная торговля» — с правом выставления заявок. Шифруется в браузере (AES-GCM + PBKDF2) и хранится только локально. Пароль шифрования не сохраняется.

Фиктивный кошелёк работает как раньше. Для T-Bank сохраните токен, затем выберите режим сверху: калькулятор спросит пароль, расшифрует токен, загрузит счёт и возьмёт депозит.

Реальный счёт Алор — депозит и торговля не подключено Развернуть

Refresh token из alor.dev → API Access Tokens. Укажите код портфеля (например D12345). Токен шифруется в браузере и хранится только локально.

Выберите брокера «Алор» сверху. Сохраните refresh token и портфель, затем режим «Торговый счёт» или «Реальная торговля».
