Сброс настроек

Спросит, сбрасывать ли также каталог логик к встроенным строкам из кода.

Объём сделки (как в роботе)

—

Deposit percent: лот = (депозит × Volume / 100) / цена. Contracts: фиксированный лот. Contract currency: Volume / цена. Максимальное количество позиций (@@MaxPos) — в блоке «Параметры управления» выше.

Комиссия: Commission, % от суммы каждой сделки (|цена × объём|); удерживается при покупке и продаже. FINRESP и % годовых в блоке выше уже с учётом комиссии.

Учёт открытых лотов (песочница)

Только для режима «Песочница (фейк)». По умолчанию FIFO (как averagePositionPriceFifo у T-Bank). Cash, открытые legs и фин. результат пересчитываются из журнала исполнений (ledger) — один источник правды, без расхождения агрегатов.

Устойчивость расчёта

Каждый пересчёт: OHLC для индикаторов и сигналов Op/Cl сдвигаются случайно в пределах 0,1%. FINRESP, комиссии и сделки — по реальным ценам MOEX.

Стакан (@OBT, live)

Та же галочка — в блоке «Реальная торговля» (реальный счёт и песочница). Одна настройка: live.obTrendConfirm. В FINRESP по истории стакана нет.

Используемые индикаторы

Отключенные индикаторы удаляются из Op/Cl строки до расчета. SL/TP позиции остаются отдельными параметрами.

По умолчанию выбраны все индикаторы. Оптимизация перебирает все непустые комбинации и оставляет набор с лучшим FINRESP.

MOEX — тикеры акций и префиксы фьючерсов

Как вкладки «MOEX акции» / «MOEX фьючерсы» в роботе. Акции: «Подобрать» — из текста поля. Фьючерсы: «Подобрать» и «Рассчитать» — контракты MOEX по префиксам и периоду расчёта: экспирация (LASTTRADEDATE) не раньше начала периода, выпуск (FIRSTTRADEDATE) внутри периода; бессрочные (…RUBF, IMOEXF) — по экспирации. При «выбрать все фьючерсы» в расчёт попадают все подходящие серии, не только по одному на префикс.

Реальный счёт T-Bank — депозит и торговля не подключено Развернуть

Токен нужен для чтения счёта и депозита; для режима «Реальная торговля» — с правом выставления заявок. Шифруется в браузере (AES-GCM + PBKDF2) и хранится только локально. Пароль шифрования не сохраняется.

Фиктивный кошелёк работает как раньше. Для T-Bank сохраните токен, затем выберите режим сверху: калькулятор спросит пароль, расшифрует токен, загрузит счёт и возьмёт депозит.
