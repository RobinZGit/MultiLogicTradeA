1. Идея

Индикатор — ряд чисел (по одному значению на бар), как многочлен: коэффициенты задают вклад на каждой позиции. Составной индикатор собирается из встроенных рядов (SMA(20), CMA(100;P=2), pp …) операциями ниже. Он сохраняется в конфиге, подсвечивается фиолетовым в списке и доступен на графике как Имя().

Математика свёртки и покомпонентного умножения — в базе знаний по многочленам; здесь — синтаксис конструктора.

1.1. Индексы ряда: младший и старший

Ряд цен pp — это многочлен по барам:

pp = (p₀; p₁; p₂; …; p_T)
  • Индекс 0 — младший: самый старый бар в истории.
  • Индекс T — старший: самая последняя цена (close текущего конца ряда); за ней в pp уже ничего нет.
  • Запись pp[k] в конструкторе — k-я цена от конца на баре t (pp[1] = close бара t, pp[5] = close бара t−4).
Свёртка pp * ww. На баре t: s_t = w₀·p_t + w₁·p_{t−1} + … + w_{N−1}·p_{t−N+1}. Последняя цена p_t (старший индекс в ряду) умножается на первый вес w₀ в окне; самая старая в окне — на последний вес w_{N−1}. Поэтому {1/20; n=1..20} даёт MA(20): первый 1/20 — вес текущего close, последний — вес 20 баров назад.

2.1. Несколько строк, переменные, комментарии

Формула может занимать несколько строк. Разделитель — перевод строки. Строки Имя = выражение задают переменные. Итоговая строка — RETURN выражение (рекомендуется) или выражение / имя = ….

Комментарии — // до конца строки (символ # занят покомпонентным умножением).

harm = {1/n; n=1..10}
// sma20: свёртка close с окном 20×(1/20) — эквивалент SMA(20)
win20 = {1/20; n=1..20}
sma20 = pp * win20
// sma50 — встроенный SMA(50)
sma50 = SMA(50)
smaLag = sma20.shift(5)
mix = sma20 # harm
RETURN mix - smaLag - sma50

{1/n; n=1..10} — сумма 1/1+…+1/10. {1/20; n=1..20} — многочлен из 20 весов 1/20 (без перечисления). .shift(5) — сдвиг ряда назад на 5 баров. pp[5] — на баре t взять цену t−4 (5-я от конца).

В переменную можно положить индикатор, цену (c-close, pp_open), число, многочлен {1;0;0;0} или выражение с операциями. Последняя строка может быть и присваиванием (gamma = alpha * beta) — тогда результатом будет правая часть.

2. Атомы (что можно писать)

РынокКоротко: pp oo hh ll vv. С дублированной первой буквой: c-close / cclose, o-open, h-high, l-low, v-volume. С префиксом pp_: pp_close, pp_open, pp_high, pp_low, pp_volume
ВстроенныеТе же формы, что в чекбоксах: SMA(100), CMA(100;P=1), LinReg(20;Dev=2), Bollinger(20;Dev=2), ATR(14), Stoch(14-3-3), MACD(12,26,9), CCI(20), Momentum(10), VWAP() и др.
Другие составныеИмя без скобок, если индикатор уже создан: Spread20_50
ЧислоЛитерал: 2, 0.05, 1,5 (запятая как десятичный разделитель)

3. Операции

+ −Сложение и вычитание рядов по барам
#Покомпонентное умножение: на каждом баре a[k] × b[k]
/#Покомпонентное деление (деление на 0 → пусто)
*Свёртка многочленов: (a*b)[k] = Σ a[i]·b[j] при i+j=k. Индекс в pp растёт со временем (старший = последний бар); в окне весов w₀ — на текущую цену, см. §1.1
( )Группировка
[k]Постфикс: k-я цена от конца на текущем баре. pp[1] — close этого бара; pp[5] — четыре бара назад
.shift(n)Сдвиг всего ряда назад на n баров: SMA(20).shift(5), sma20.shift(3)

4. Фигурные скобки { … }

4.1. Одно число — константа

{1000}

Скаляр 1000 на всех барах. Умножение: pp # {2} — удвоить цену покомпонентно.

4.2. Список коэффициентов — многочлен

{1;0;0;0}

Многочлен с коэффициентами 1, 0, 0, 0 (единица на позиции 0). Эквивалент сдвига «умножить на x³» в дискретной модели. Дроби через слэш:

{1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20}

То же окно короче через диапазон (20 равных весов 1/20):

{1/20; n=1..20}

Свёртка с pp даёт простое MA(20):

pp * {1/20; n=1..20}
Важно: {1000} — одно число; {1;0;0;0} — четыре коэффициента (многочлен). Не путать с записью «один и нули» через точку с запятой.

4.3. Сумма по индексу

{ SMA(n); n=5..30 }

Сумма SMA(5)+SMA(6)+…+SMA(30). Альтернатива границ: n0=5; n1=30. Переменная в формуле члена должна совпадать с именем в n=….

{1/n; n=1..10}

Гармоническая сумма 1/1+1/2+…+1/10 — скаляр на каждом баре; удобно масштабировать ряд через # (см. пример в §2.1).

{1; n=1..100}

Сумма ста единиц — константа 100 на каждом баре (после свёртки/сложения с рядом даёт масштабирование).

5. Примеры формул

Спред двух SMA

SMA(20) - SMA(50)

Классический «спред» скользящих; имя в конструкторе, например Spread20_50.

Отклонение от SMA

(pp - SMA(20)) # 2

Удвоенное расстояние цены от SMA(20) на каждом баре.

Цена: индекс от конца и сдвиг ряда

pp[5]

На баре t — close бара t−4 (5-я цена от конца, считая текущий бар как 1-ю).

pp.shift(5)

Весь ряд close сдвинут назад на 5 баров: на баре t значение с бара t−5.

Свёртка цены с окном

pp * {1/20; n=1..20}

Сумма SMA по диапазону

{ SMA(n); n=5..30 }

Комбинированный пример (многострочный)

// MA(20) через свёртку
win20 = {1/20; n=1..20}
pp * win20

fast = SMA(20)
slow = SMA(50)
spread = fast - slow
spread

// или: close # (close - SMA(20))  — отклонение от SMA

6. Интерфейс

  • + — диалог: имя (латиница, цифры, _ . -) и формула. OK — индикатор добавляется с включённой галочкой и фиолетовой подсветкой.
  • i. — эта справка.
  • На графике инструмента в «Добавить индикатор» введите Имя() (параметры в скобках не нужны).
  • Составные пока используются в расчёте FINRESP только если подключены в строках логики вручную; основной сценарий — визуализация на графике и хранение в конфиге.

7. Ограничения

  • Имя должно быть уникальным среди составных индикаторов.
  • В формуле доступны только индикаторы из списка чекбоксов (встроенные формы с параметрами).
  • Циклические ссылки между составными индикаторами не допускаются.
  • Строки, начинающиеся с //, — комментарии; пустые строки игнорируются.
