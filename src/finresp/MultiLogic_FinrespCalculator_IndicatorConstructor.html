Конструктор составных индикаторов

Калькулятор FINRESP · блок «Используемые индикаторы» · кнопки + и i.

← Калькулятор · Справка калькулятора · Многочлены и операции * #

1. Идея

Индикатор — ряд чисел (по одному значению на бар), как многочлен: коэффициенты задают вклад на каждой позиции. Составной индикатор собирается из встроенных рядов (SMA(20), CMA(100;P=2), pp …) операциями ниже. Он сохраняется в конфиге, подсвечивается фиолетовым в списке и доступен на графике как Имя().

Математика свёртки и покомпонентного умножения — в базе знаний по многочленам; здесь — синтаксис конструктора.

2. Атомы (что можно писать)

Рынокpp close, oo open, hh high, ll low, vv volume
ВстроенныеТе же формы, что в чекбоксах: SMA(100), CMA(100;P=1), LinReg(20;Dev=2), Bollinger(20;Dev=2), ATR(14), Stoch(14-3-3), MACD(12,26,9), CCI(20), Momentum(10), VWAP() и др.
Другие составныеИмя без скобок, если индикатор уже создан: Spread20_50
ЧислоЛитерал: 2, 0.05, 1,5 (запятая как десятичный разделитель)

3. Операции

+ −Сложение и вычитание рядов по барам
#Покомпонентное умножение: на каждом баре a[k] × b[k]
/#Покомпонентное деление (деление на 0 → пусто)
*Свёртка многочленов: (a*b)[k] = Σ a[i]·b[j] при i+j=k. Имитация «фильтра» / скользящего окна
( )Группировка
[n]Сдвиг: как фактор [5] — дельта на позиции 5; свёртка [5] * pp сдвигает цену на 5 баров. Постфикс pp[5] или SMA(20)[5] — сдвиг уже собранного ряда

4. Фигурные скобки { … }

4.1. Одно число — константа

{1000}

Скаляр 1000 на всех барах. Умножение: pp # {2} — удвоить цену покомпонентно.

4.2. Список коэффициентов — многочлен

{1;0;0;0}

Многочлен с коэффициентами 1, 0, 0, 0 (единица на позиции 0). Эквивалент сдвига «умножить на x³» в дискретной модели. Дроби через слэш:

{1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20}

Нормированное окно из 20 равных весов (свёртка с pp даёт простое MA(20)):

pp * {1/20;1/20;…;1/20}
Важно: {1000} — одно число; {1;0;0;0} — четыре коэффициента (многочлен). Не путать с записью «один и нули» через точку с запятой.

4.3. Сумма по индексу

{ SMA(n); n=5..30 }

Сумма SMA(5)+SMA(6)+…+SMA(30). Альтернатива границ: n0=5; n1=30. Переменная в формуле члена должна совпадать с именем в n=….

{1; n=1..100}

Сумма ста единиц — константа 100 на каждом баре (после свёртки/сложения с рядом даёт масштабирование).

5. Примеры формул

Спред двух SMA

SMA(20) - SMA(50)

Классический «спред» скользящих; имя в конструкторе, например Spread20_50.

Отклонение от SMA

(pp - SMA(20)) # 2

Удвоенное расстояние цены от SMA(20) на каждом баре.

Цена со сдвигом

[5] * pp

Close пять баров назад, выровненный по текущей шкале времени.

Свёртка цены с окном

pp * {1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20}

Сумма SMA по диапазону

{ SMA(n); n=5..30 }

Комбинированный пример (как в placeholder формы)

SMA(20) - SMA(50)
pp * {1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20}
(pp - SMA(20)) # 2
[5] * pp
{ SMA(n); n=5..30 }
{1; n=1..100} * {1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20;1/20}

6. Интерфейс

  • + — диалог: имя (латиница, цифры, _ . -) и формула. OK — индикатор добавляется с включённой галочкой и фиолетовой подсветкой.
  • i. — эта справка.
  • На графике инструмента в «Добавить индикатор» введите Имя() (параметры в скобках не нужны).
  • Составные пока используются в расчёте FINRESP только если подключены в строках логики вручную; основной сценарий — визуализация на графике и хранение в конфиге.

7. Ограничения

  • Имя должно быть уникальным среди составных индикаторов.
  • В формуле доступны только индикаторы из списка чекбоксов (встроенные формы с параметрами).
  • Циклические ссылки между составными индикаторами не допускаются.
  • Строки, начинающиеся с # в поле формулы, при проверке считаются комментариями (в placeholder).
